E-Commerce Website

Project Overview

This project is a fully responsive E-Commerce website developed using HTML, CSS, and Bootstrap. The website provides a simple and user-friendly interface for customers to browse products, view details, and manage their shopping cart. It also includes an Admin Dashboard to manage categories and products. The design focuses on clean layout, responsiveness, and smooth user experience across different devices.

Pages Implemented

Home Page: Attractive landing page with featured products and Categories.

Login Page: User authentication interface for existing users.

Sign Up Page: Registration page for new users.

Shop Page: Displays available products in a grid layout.

Category Pages: Products organized according to categories.

Product Detail Page: Detailed view of product including image, description, and price.

Cart Page: Shows selected items with quantity and total price.

Admin Dashboard: Interface for managing products and stats.

Technologies Used

HTML5: For structuring the website.

CSS3: For styling and layout design.

Bootstrap 5: For responsive design and modern UI components.
